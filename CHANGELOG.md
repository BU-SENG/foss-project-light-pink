# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-15

### Added
- Initial release of AI Docstring Generator
- Python code parser with function, class, and method detection
- JavaScript/TypeScript code parser
- Integration with Gemini 2.0 Pro AI API
- Multiple docstring format support (Google, NumPy, Sphinx, JSDoc)
- Monaco Editor for code editing and preview
- Drag & drop file upload
- Supabase authentication (email, GitHub, Google OAuth)
- History tracking for logged-in users
- Offline mode with localStorage
- Responsive UI with TailwindCSS
- Dark mode support
- Real-time docstring generation
- File download functionality
- Row-Level Security (RLS) with Supabase
- Serverless Edge Functions for AI processing

### Documentation
- Comprehensive README with setup instructions
- Detailed Supabase setup guide
- Contributing guidelines
- MIT License

### Technical
- React 18 with TypeScript
- Vite build system
- Supabase backend
- PostgreSQL database
- Deno Edge Functions
- TailwindCSS styling
- React Router for navigation
- Monaco Editor integration
- Lucide icons
